# machine_vision
FInal Project of Machine Vision 2021
by Sugiarto Wibowo

1. Install Pycharm and Arduino IDE
2. Install cv2, pyserial on pycharm
3. Inclide servo


Components:
1. Arduino UNO
2. Webcam
3. Servo HS-645 MG
4. Power supply 5v
